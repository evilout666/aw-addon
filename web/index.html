General & Deployment

The Afterwork cog is a unified administrative system designed for e.Network server operations. It runs on a prefix-based architecture (default prefix is represented as [p]). All primary commands are group commands and require Bot Owner permissions.

Base Commands
[p]afterwork Owner Only
Displays the System Status Dashboard, showing real-time configuration status, monitored feeds, active voice rooms, and visibility settings for all modules.
[p]afterwork help Owner Only
Lists all subcommands registered under the Afterwork command group with inline syntax help.
Deployment Commands

Deployment commands spawn persistent Discord Embed + Button/Select Menu panels that allow server staff or users to interact with modules without having to type commands.

[p]afterwork deploy Owner Only
Sequentially deploys and pins all 6 module management panels in the current channel. Any previously deployed panels will be deleted to keep channels clean.
[p]afterwork deploy [module] Owner Only
Deploys/pins a specific module's management panel in the current channel.
Parameter Type Description
module string Must be one of: audio, embed, rss, tv, voice, hide.

Audio Module

Integrates Red-DiscordBot's Audio system into a single persistent playback terminal. Provides a web-dashboard-like control station inside Discord for playing predefined music streams.

Commands
[p]afterwork deploy audio Owner Only
Spawns the music configuration panel and playback controls message.
Interactive Panel Functions
Enable / Disable System

Toggles the bot's auto-play service. When active, the bot joins the specified voice channel and begins playing chosen streams.

Set Music Channel

Opens a modal where you paste a Voice Channel ID to bind the music bot's streaming target.

Add/Remove Playlist

Spawns a modal to add/remove YouTube/Spotify URLs cataloged under custom names for dropdown playback selection.

Playback Control Bar

Buttons to instantly Pause, Resume, Skip current tracks, or Stop & Disconnect the bot from the music channel.

Embed Module

Allows administrators to design and publish beautiful rich embeds to announcements or general channels using custom JSON structures generated directly from this dashboard.

Commands
[p]afterwork deploy embed Owner Only
Deploys the persistent Embed Shortcuts panel to manage named channel aliases.
[p]afterwork embed [channel] [json] Owner Only
Publishes an embed formatted by the JSON string parameter to the specified channel (accepts channel mention, ID, or saved name).
Parameter Type Description
channel string / mention Target channel ID, mention, or a configured channel name alias.
json JSON Block The payload specifying fields like: title, description, color, thumbnail, image, and fields.
Interactive Panel Functions
Add Channel Alias

Opens a modal to register a shortcut text key (e.g. news) mapped to a specific channel ID, making embed distribution faster.

Remove Alias

Prompts a modal to delete a saved channel shortcut key from the bot's configuration storage.

RSS Module

Monitors external RSS or Atom feeds (e.g. gaming blogs, developer news, github releases) in the background and prints new posts in specific channels automatically.

Commands
[p]afterwork deploy rss Owner Only
Spawns the RSS settings control panel.
[p]afterwork rss add [feed_name] [channel] [url] Owner Only
Subscribes a channel to a news feed. Automatically detects and parses incoming posts.
Parameter Type Description
feed_name string A unique slug label to identify this feed configuration.
channel channel mention Discord channel where feed alerts should be printed.
url string (URL) Valid RSS or Atom XML link.
[p]afterwork rss remove [feed_name] Owner Only
Deletes an active RSS/Atom feed subscription from the tracking list.
Parameter Type Description
feed_name string The name slug assigned to the subscription.
Interactive Panel Functions
Toggle Feeds Loop

Toggles the background scraper. Enables or pauses background checks for all feed subscriptions globally without losing feed data.

TV Module (Sonarr/Radarr Alerts)

Creates specialized webhook integration endpoints for Radarr and Sonarr instances to notify server users when new movies or TV episodes are released, formatting notifications cleanly.

Commands
[p]afterwork deploy tv Owner Only
Deploys the TV alerts settings panel, providing instructions and endpoints for webhooks.
Interactive Panel Functions
Enable Webhooks

Toggles Sonarr/Radarr webhook event parsing. When disabled, the bot rejects movie/show alert triggers.

Set Alert Channel

Opens a text input modal to set the channel ID where all media notifications are sent.

Endpoint Integrations

Processes incoming payloads to render cards with poster images, season info, and direct links dynamically.

Voice Module (AutoRoom Controllers)

Synchronizes with the server's AutoRoom cog to let users who join a generator voice channel create their own temporary rooms, giving creators personal moderation controls inside Discord.

Commands
[p]afterwork deploy voice Owner Only
Deploys the main Voice Module settings controller.
Interactive Panel Functions
Toggle AutoRoom Sync

Enables or disables tracking of temporary rooms created by AutoRoom. Essential for active controls.

Set Join Channel ID

Opens a modal to configure the generator voice channel (e.g. ➕︱CREATE) which users join to trigger room creation.

Room Owner Controls

When a temporary room is active, a control panel message provides the room owner buttons to:
• Make Private / Public (toggle default connection permission)
• Transfer Ownership (demote owner/promote active member)
• Kick Member (force disconnect user and block connection)

Hide Module

Enables a toggle to hide or show an entire category and all text/voice channels contained within it from administrator and channel-manager roles, keeping admin operations private during development or maintenance.

Commands
[p]afterwork deploy hide Owner Only
Deploys the category visibility panel with Category ID setter and the Show / Hide toggle.
Interactive Panel Functions
Category ID Modal

Spawns a text input modal to paste the category channel ID to bind as the target of the visibility toggle.

Show / Hide Button

Instantly alters permissions overwrites for the target category channel and all contained text/voice channels for roles holding Administrator or Manage Channels permissions:
• Hide: Denies view_channel override.
• Show: Resets view_channel override to neutral.
